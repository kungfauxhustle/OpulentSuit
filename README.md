## Contents:

Enclosed are a set of Opulent Suits for the "Great Great Asset To The Company (TM)" and co.  

Package Includes:
* Ruby Red
* Orange Citrine
* Heliodor Yellow
* Emerald Green 
* Blue Sapphire
* Purple Amethyst
* Pink Garnet
* Onyx Black
* White Diamond
